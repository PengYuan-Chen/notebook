服務器搭建
- 利用 Node.js 的 express 框架建立
- npm : ejs、express、mongoose、lowdb、md5、shortid、cookie-parser
- MongoDB 儲存用戶資料
- cookie、sessions 身分校驗
- Postman 模擬請求返回 json 數據 

功能
- 用戶註冊後即可登入
- 登入後享有新增、修改、刪除清單等功能
- 未登入則不能進行任何訪問
